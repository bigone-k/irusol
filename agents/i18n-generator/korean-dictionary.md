# 한국어 순수 우리말 사전

## 원칙
1. **순수 우리말 우선**: 외래어나 한자어 대신 순수 우리말 사용
2. **자연스러움**: 억지스럽지 않고 자연스러운 표현
3. **현대적 감각**: 현대 한국어 사용자가 이해하기 쉬운 표현

## 사용 예시

### ❌ 나쁜 예
```json
{
  "task": {
    "create": "태스크 생성",
    "list": "태스크 리스트"
  }
}
```

### ✅ 좋은 예
```json
{
  "task": {
    "create": "작업 만들기",
    "list": "작업 목록"
  }
}
```

## 특수 규칙

### 1. 복합어 처리
- "작업 만들기" (O) vs "작업만들기" (X)
- 띄어쓰기로 가독성 향상

### 2. 존댓말 사용
- 사용자 대상 메시지는 존댓말 사용
- 예: "작업을 만들어주세요", "저장하시겠습니까?"

### 3. 조사 처리
- 받침에 따라 적절한 조사 사용
- "작업을" (받침 O), "목표를" (받침 O)
- "레벨이" (받침 X), "경험치가" (받침 X)

### 4. 문맥 고려
- 같은 단어도 문맥에 따라 다른 표현 사용
- 예: "complete" → "완료하기" (동작), "완료됨" (상태)
