# 개발지시서 — MVP 템플릿(=모듈) 자동 생성

## 5) 템플릿 스펙(확정본) — A~D

> 표기: `D#`는 “생성일(오늘) + #일”에 해당  
> Habit 빈도는 기본을 “특정 요일”로 두되, 사용자가 바꿀 수 있어야 함.

---

### 템플릿 A — 건강(2kg 감량)
#### Goal
- title: `[3개월] 2kg 감량하기`
- current: 0
- target: 2
- unit: `kg`
- description: `꾸준한 운동 습관을 만들고 3개월 내 2kg 감량을 달성합니다.`
- start/end: 자동(3개월)

#### Project
- title: `[1개월] 운동 습관 만들기`
- description: `1개월 동안 ‘주 3회 운동’을 습관화하는 데 집중합니다.`
- start/end: 자동(1개월)

#### Quests
- TASK D0: `나에게 맞는 운동 유형 정하기(예: 헬스/요가/러닝 등)`
- TASK D1: `운동 등록하기(예: 헬스장 등록/요가원 등록/홈트 프로그램 선택 등)`
- TASK D2: `운동 스케줄 정하기(예: 퇴근 후 바로 헬스장 가기)`
- HABIT: `주 3회 운동하기`
  - frequency default: mode=BY_WEEKDAY, weekdays=[Mon,Wed,Fri] (예시 기본값)

### 템플릿 B — 마음건강(취향리스트 → 10회 실행)
#### Goal
- title: `[3개월] 취향리스트 완성 후 10회 실행하기`
- current: 0
- target: 10
- unit: `회`
- description: `취향리스트를 만들고, 10회 실행합니다.`
- start/end: 자동(3개월)

#### Project
- title: `[1개월] 취향리스트 만들기`
- description: `나를 기분 좋게 하는 취향 리스트를 만들고 실행합니다.`
- start/end: 자동(1개월)

#### Quests (전부 TASK)
- TASK D0: `최근 기분 좋았던 경험 3개 적기`
- TASK D1: `내가 좋아하는 것 20개 이상 적어보기`
- TASK D3: `리스트를 카테고리로 묶고 부족한 부분 채워보기(예: 음식/장소/사람 등)`
- TASK D4: `리스트에서 이번 달 ‘해보기’ 후보 5개 고르기`
- TASK D5: `후보 5개 중 2개를 확정하여 캘린더에 저장하기(언제/어디서/무엇을)`
- TASK D16: `1개 실행하고 만족도와 기분 기록하기`
- TASK D25: `1개 더 실행하고 만족도/기분 기록하기`
- TASK D27: `취향리스트 완성하기`

### 템플릿 C — 독서(3권 완독)
#### Goal
- title: `[3개월] 책 3권 완독하기`
- current: 0
- target: 3
- unit: `권`
- description: `책 읽는 습관을 만들고 책 3권을 완독합니다.`
- start/end: 자동(3개월)

#### Project
- title: `[1개월] 독서 루틴 만들기`
- description: `독서 환경을 세팅하고 꾸준히 읽는 습관을 만듭니다.`
- start/end: 자동(1개월)

#### Quests
- TASK D0: `독서 가능 시간 정하기(예: 출퇴근 지하철/자기 전 등)`
- TASK D0: `읽을 책 1권 고르기`
- HABIT: `주 4회 5분 이상 독서`
  - frequency default: mode=TIMES_PER_WEEK, times_per_week=4  (요일 자유가 UX에 더 자연스러움)
- TASK D25: `책 1권 완독`
- TASK D27: `책 1권 감상평 1줄 써보기`

### 템플릿 D — 생활(집밥 레시피 10개 모으기)
#### Goal
- title: `[3개월] 나만의 집밥 레시피 10개 모으기`
- current: 0
- target: 10
- unit: `개`
- description: `배달·외식을 줄이고 나를 위한 집밥 레시피를 10개 모아 정리합니다.`
- start/end: 자동(3개월)

#### Project
- title: `[1개월] 집밥 루틴 만들기 (주 2회)`
- description: `건강한 레시피를 찾고 집밥을 하는 습관을 만듭니다.`
- start/end: 자동(1개월)

#### Quests
- HABIT: `주 2회 집밥 1끼 해먹기`
  - frequency default: mode=TIMES_PER_WEEK, times_per_week=2
- TASK D0: `건강하고 맛있는 레시피 10개 저장하기`
- TASK D3: `가장 쉬운 레시피 1개 해보기`
- TASK D7: `레시피 1개를 내 입맛에 맞게 변형하고 ‘나만의 레시피’로 기록하기`
